# TodoManagement

This repository contains a Spring Boot-based Todo Management application. The goal of the project is to develop a scalable backend application that enables users to efficiently organize their projects, tasks, and daily activities.

## Tech stack
- Java 21
- Spring Boot 4.x (Spring Data JPA, Spring Web MVC, Validation)
- H2 (in-memory) for development
- SpringDoc OpenAPI (swagger)
- JWT 
- PostgreSQL 
- Hibernate
- Lombok
- Maven
- JUnit & Mockito 
- Docker
- Kubernetes
- GitHub Actions (CI/CD)
